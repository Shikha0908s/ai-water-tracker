. AI Water Tracker 

This is a simple AI-based web app that helps you track your daily water intake and gives smart suggestions to stay hydrated.

You can enter how many glasses of water you drank, and the app will store it and generate personalized feedback using a local AI model (no API needed).

> Features
- Log daily water intake
- Stores data using SQLite
- Gives AI-based hydration suggestions
- Works completely offline (local AI)

> Tech Used
- Python (Flask)
- SQLite
- GPT4All (local AI model)

# Run the App
```bash
conda activate water-tracker
python app.py
